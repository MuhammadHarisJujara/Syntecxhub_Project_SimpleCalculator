# Simple Calculator (Python)

## Project Overview
This is a command-line based Simple Calculator built using Python.  
It was developed as part of my internship at SyntecxHub.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Divide-by-zero error handling
- Modular functions for clean code structure
- Menu-driven interface for repeated calculations

---

## How to Run
1. Install Python on your system
2. Download or clone this repository
3. Open terminal in project folder
4. Run the file using:

```bash
python calculator.py
